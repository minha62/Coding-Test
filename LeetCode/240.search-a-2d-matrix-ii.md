## 풀이2: 파이썬다운 방식

파이썬의 `any()` 함수를 사용하면 내부적으로 행렬에 값이 존재하는지 여부를 위에서부터 차례대로 한 줄씩 탐색 가능

```python
def searchMatrix(self, matrix, target):
  return any(target in row for row in matrix)
```
